<h1>MGNREGA : RURAL EMPLOYMENT  MANAGEMENT DATA</h1>
<p>MGNREGA(The Mahatma Gandhi National Rural Employment Guarantee Act ) is a scheme by which unskilled people of rural India are guaranteed to have 100 days of paid work. Main objective of this project is to manage employments offered through this scheme. There are two users of this system</p>
<ul>
  <li>a. BDO (Block Development Officer)</li>

   <li>b. Gram Panchayat member</li>
</ul>
<p>A project is created by BDO. Project is where people are assigned to. Before assigning anybody to a project(done by Gram Panchayat member) his/her data must be inserted into the system.</p>
<br>
<h4>The MGNREGA is a java application to provide all information of BDO and Gram Panchayat Member (GPM). This project very easy and better user experience. this project is also assigning and managing the projects and employees . It will stores Gram Panchayat Member(GPM), Employee and project's data assignments, project record's, wage/day and more informations have in this project. <h2>It is Console based Project.</h2></h4>
